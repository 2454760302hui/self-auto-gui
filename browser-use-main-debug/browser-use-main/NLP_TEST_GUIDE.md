async def test_custom_scenario(self):

    logger.info("\n" + "="*60)
    logger.info("🎯 测试自定义场景")
    logger.info("="*60)
    
    try:
        logger.info("  🔹 自定义操作...")
        # await self.agent.act("你的自定义命令")
        logger.info("  ✅ 自定义操作成功")
        self.test_results['custom_操作'] = '✅ 通过'
    except Exception as e:

await self.test_custom_scenario()

## 🐛 故障排除

### 问题1: 测试页面无法打开

**症状**: 无法打开 test_page.html

**解决方案**:
1. 检查文件路径是否正确
2. 确保文件存在
3. 尝试使用绝对路径

```python
test_page_path = Path(__file__).parent / "test_page.html"
print(f"测试页面路径: {test_page_path.absolute()}")
```

### 问题2: 元素无法找到

**症状**: 点击或填充元素失败

**解决方案**:
1. 检查元素是否存在
2. 检查元素是否可见
3. 检查元素是否被其他元素遮挡
4. 使用 NLP 描述元素位置

```python
# 使用更具体的描述
await agent.act("点击标题为'测试按钮'的蓝色按钮")
```

### 问题3: iframe 操作失败

**症状**: 无法进入或操作 iframe

**解决方案**:
1. 确保 iframe 已加载
2. 使用正确的 iframe 选择器
3. 等待 iframe 内容加载

```python
# 等待 iframe 加载
await agent.act("等待iframe加载完成")
# 进入 iframe
await agent.act("进入iframe")
```

### 问题4: 弹窗处理失败

**症状**: 无法处理弹窗

**解决方案**:
1. 确保弹窗已显示
2. 使用正确的弹窗处理命令
3. 检查弹窗类型

```python
# 处理不同类型的弹窗
await agent.act("接受alert弹窗")
await agent.act("确认confirm弹窗")
await agent.act("在prompt弹窗中输入 内容")
```
