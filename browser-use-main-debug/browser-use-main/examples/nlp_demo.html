🎯 意图识别

识别用户输入的自动化意图

📝 示例命令

点击登录按钮
输入用户名和密码
等待页面加载完成
选择下拉菜单中的选项

识别结果

📊 支持的意图类型

  • CLICK - 点击操作
  • INPUT - 输入操作
  • SELECT - 选择操作
  • SUBMIT - 提交操作
  • WAIT - 等待操作
  • VERIFY - 验证操作
  • NAVIGATE - 导航操作
  • EXTRACT - 提取操作

🏷️ 实体提取

从文本中提取关键实体信息

📝 示例文本

点击ID为submit的按钮
在class为input-field的输入框输入密码
选择name为category的下拉菜单中的第一个选项

提取结果

🔍 实体类型

  • ELEMENT - 页面元素
  • SELECTOR - CSS选择器
  • VALUE - 输入值
  • ATTRIBUTE - 属性
  • CONDITION - 条件
  • LOCATION - 位置信息
  • TIMING - 时间信息

⚙️ 命令解析

将自然语言转换为结构化命令

等待结果加载，然后点击第一个结果

解析结果

📋 命令类型

  • CLICK - 点击元素
  • TYPE - 输入文本
  • SELECT - 选择选项
  • SUBMIT - 提交表单
  • WAIT - 等待条件
  • VERIFY - 验证状态
  • NAVIGATE - 页面导航

📝 DSL解析

解析特定领域语言（DSL）命令

CLICK #submit-btn
INPUT .username-field "admin"
WAIT 3000 | CLICK #next
SELECT [name=category] "option1"

解析结果

🔧 DSL操作符

  • CLICK - 点击操作
  • INPUT - 输入操作
  • SELECT - 选择操作
  • SUBMIT - 提交操作
  • WAIT - 等待操作
  • VERIFY - 验证操作
  • | - 管道操作符
