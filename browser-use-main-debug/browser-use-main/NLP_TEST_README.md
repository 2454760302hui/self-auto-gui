## 🐛 故障排除

### 问题1: 测试页面无法打开

**解决方案**:
1. 检查文件路径
2. 确保文件存在
3. 使用绝对路径

### 问题2: 元素无法找到

**解决方案**:
1. 检查元素是否存在
2. 检查元素是否可见
3. 使用更具体的描述

### 问题3: iframe 操作失败

**解决方案**:
1. 确保 iframe 已加载
2. 使用正确的选择器
3. 等待 iframe 内容加载

### 问题4: 弹窗处理失败

**解决方案**:
1. 确保弹窗已显示
2. 使用正确的处理命令
3. 检查弹窗类型

详见 [NLP_TEST_GUIDE.md](NLP_TEST_GUIDE.md#故障排除)
