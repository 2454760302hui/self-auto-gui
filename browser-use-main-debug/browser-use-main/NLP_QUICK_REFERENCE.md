## 💡 提示

### 1. 使用自然语言

✅ 好的做法:
```
点击那个蓝色的提交按钮
```

❌ 不好的做法:
```
点击 button#submit
```

### 2. 提供足够的上下文

✅ 好的做法:
```
在登录表单中填充用户名为 user@example.com
```

❌ 不好的做法:
```
填充为 user@example.com
```

### 3. 使用明确的值

✅ 好的做法:
```
选择日期为 2024-05-06
```

❌ 不好的做法:
```
选择日期为 今天
```

### 4. 等待必要的时间

✅ 好的做法:
```
点击按钮
等待网络空闲
获取结果
```

❌ 不好的做法:
```
点击按钮
立即获取结果
```

### 5. 验证操作结果

✅ 好的做法:
```
点击提交按钮
等待页面加载
检查是否显示成功提示
```

❌ 不好的做法:
```
点击提交按钮
```
