## 🔧 新增操作详情

### 1️⃣ 键盘操作 (8 个)

```
✅ 输入文本 (TYPE_TEXT)
   - 自动输入文本到输入框
   - 触发 input 事件

✅ 按 Enter 键 (PRESS_ENTER)
   - 模拟 Enter 键按下
   - 用于表单提交

✅ 按 Tab 键 (PRESS_TAB)
   - 模拟 Tab 键按下
   - 用于元素导航

✅ 按 Escape 键 (PRESS_ESCAPE)
   - 模拟 Escape 键按下
   - 用于关闭弹窗

✅ Ctrl+C 复制 (CTRL_C)
   - 模拟 Ctrl+C 快捷键
   - 复制选中文本

✅ Ctrl+V 粘贴 (CTRL_V)
   - 模拟 Ctrl+V 快捷键
   - 粘贴剪贴板内容

✅ Ctrl+A 全选 (CTRL_A)
   - 模拟 Ctrl+A 快捷键
   - 全选文本

✅ Shift+Tab 反向导航 (SHIFT_TAB)
   - 模拟 Shift+Tab 快捷键
   - 反向导航元素
```

### 2️⃣ 鼠标操作 (6 个)

```
✅ 鼠标移动 (MOVE)
   - 模拟鼠标移动事件
   - 支持坐标指定

✅ 鼠标按下 (DOWN)
   - 模拟鼠标按下事件
   - 支持按钮指定

✅ 鼠标释放 (UP)
   - 模拟鼠标释放事件
   - 支持按钮指定

✅ 鼠标滚轮 (WHEEL)
   - 模拟鼠标滚轮事件
   - 支持滚动距离指定

✅ 拖拽操作 (DRAG_DROP)
   - 模拟拖拽操作
   - 支持 dragstart 和 dragend 事件

✅ 长按操作 (LONG_PRESS)
   - 模拟长按操作
   - 默认持续 1 秒
```

### 3️⃣ 文本操作 (7 个)

```
✅ 获取文本 (GET)
   - 获取元素的文本内容
   - 返回文本值

✅ 设置文本 (SET)
   - 设置元素的文本内容
   - 触发 change 事件

✅ 清空文本 (CLEAR)
   - 清空元素的文本内容
   - 触发 change 事件

✅ 选中文本 (SELECT)
   - 选中元素的所有文本
   - 用于复制操作

✅ 复制文本 (COPY)
   - 复制文本到剪贴板
   - 使用 execCommand('copy')

✅ 追加文本 (APPEND)
   - 在文本末尾追加内容
   - 触发 change 事件

✅ 替换文本 (REPLACE)
   - 替换文本中的指定内容
   - 触发 change 事件
```
