## 1. 设计系统

## 2. 布局设计

### 2.1 整体布局结构

```
┌─────────────────────────────────────────────────────────┐
│  顶部导航栏 (h-16)                                       │
│  - Logo / 产品名称 | 设备状态 | 主题切换 | 帮助          │
├──────────────────┬──────────────────────────────────────┤
│                  │                                      │
│  左侧面板        │  中央工作区                          │

│                  │                                      │
│  - 设备列表      │  ┌──────────────┬──────────────┐    │
│  - 快速操作      │  │ 投屏显示     │ 录屏面板     │    │
│  - 功能导航      │  │ (50%)        │ (50%)        │    │
│  - 设置          │  │              │              │    │
│                  │  └──────────────┴──────────────┘    │
│                  │                                      │
│                  │  ┌──────────────────────────────┐   │
│                  │  │ 代码编辑区                   │   │
│                  │  │ (Pytest / YAML)              │   │
│                  │  └──────────────────────────────┘   │
│                  │                                      │
├──────────────────┴──────────────────────────────────────┤
│  底部面板 (h-32)                                         │
│  - AI 助手 / 日志 / 执行结果                             │
└─────────────────────────────────────────────────────────┘
```

### 2.2 响应式设计

#### 桌面版（> 1400px）
- 左侧面板：256px
- 中央工作区：flex-1
- 投屏与录屏：50% / 50%

#### 平板版（768px - 1400px）
- 左侧面板：可折叠
- 中央工作区：全宽
- 投屏与录屏：垂直堆叠

#### 手机版（< 768px）
- 不支持（仅桌面应用）

## 4. 交互设计

### 4.1 操作反馈

#### 点击操作
- 显示红色圆圈（500ms）
- 圆圈逐渐消失

#### 滑动操作
- 显示起点和终点
- 绘制滑动轨迹箭头
- 箭头逐渐消失

#### 长按操作
- 显示脉冲圆圈
- 持续 1-2 秒

#### 文本输入
- 显示输入框
- 输入内容实时显示

### 4.2 状态指示

#### 设备状态
- 在线：绿色圆点 + "已连接"
- 离线：灰色圆点 + "已断开"
- 连接中：黄色圆点 + "连接中..."

#### 操作状态
- 待执行：灰色
- 执行中：蓝色 + 加载动画
- 成功：绿色 + 勾号
- 失败：红色 + 叉号

#### 录屏状态
- 未开始：灰色按钮
- 录制中：红色按钮 + 脉冲动画
- 已停止：绿色按钮

### 4.3 动画效果

#### 页面过渡
- 淡入淡出：200ms
- 滑动：300ms

#### 按钮交互
- Hover：背景色变化 + 阴影增强
- Click：缩放 0.95 + 阴影减弱
- 加载：旋转动画

#### 列表动画
- 项目进入：从左滑入 + 淡入
- 项目删除：淡出 + 向右滑出

## 5. 页面流程

### 5.1 首页流程

```
1. 应用启动
   ↓
2. 检测设备连接
   ↓
3. 显示设备列表
   ↓
4. 用户选择设备
   ↓
5. 显示投屏画面
   ↓
6. 用户可进行操作
```

### 5.2 录屏流程

```
1. 点击"开始录屏"
   ↓
2. 执行操作序列
   ↓
3. 实时显示操作记录
   ↓
4. 点击"停止录屏"
   ↓
5. 自动生成代码
   ↓
6. 显示代码预览
   ↓
7. 用户可复制/执行
```

### 5.3 代码执行流程

```
1. 选择代码类型（Pytest/YAML）
   ↓
2. 点击"执行"
   ↓
3. 显示执行进度
   ↓
4. 实时显示执行结果
   ↓
5. 显示成功/失败状态
```

## 7. 响应式设计

### 7.1 断点

```
- 超小屏：< 640px
- 小屏：640px - 1024px
- 中屏：1024px - 1400px
- 大屏：> 1400px
```

### 7.2 布局调整

#### 中屏（1024px - 1400px）
- 左侧面板：可折叠
- 投屏与录屏：垂直堆叠
- 代码编辑：全宽

#### 小屏（640px - 1024px）
- 左侧面板：隐藏（可通过菜单打开）
- 投屏全屏显示
- 录屏面板：浮窗
- 代码编辑：隐藏（可通过标签页切换）

---

## 8. 无障碍设计

### 8.1 键盘导航
- Tab 键切换焦点
- Enter 键激活按钮
- Esc 键关闭弹窗

### 8.2 屏幕阅读器
- 所有按钮有 aria-label
- 图标有 alt 文本
- 表单有 label 关联

### 8.3 色彩对比
- 文本与背景对比度 ≥ 4.5:1
- 界面元素对比度 ≥ 3:1

---

## 9. 性能指标

### 9.1 加载性能
- 首屏加载：< 2s
- 投屏首帧：< 500ms
- 代码生成：< 2s

### 9.2 交互性能
- 点击响应：< 100ms
- 滑动帧率：60fps
- 动画帧率：60fps

### 9.3 资源优化
- 图片压缩：WebP 格式
- 代码分割：按需加载
- 缓存策略：合理使用缓存
